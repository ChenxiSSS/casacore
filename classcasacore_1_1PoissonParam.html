Detailed Description

template<class T>
class casacore::PoissonParam< T >

A one dimensional Poisson function.

Intended use:

Internal

Review Status

Test programs:
tFunctionHolder

Prerequisite

  • FunctionParam class
  • Function class

Etymology

A 1-dimensional Poisson.

Synopsis

A Poisson is described by lambda The value is:

height (|x-center| == 0.0)
0 (|x-center| != 0.0)

The parameters are enumerated by LAMDA. They have default values of 1.

Example

PoissonFunction<Double> sf(5.0);
sf(25); // = 5.0

Template Type Argument Requirements (T)

  • T should have standard numerical operators

Thrown Exceptions

  • AipsError if incorrect parameter number specified.
