Detailed Description

template<class T>
class casacore::OddPolynomial< T >

A one dimensional odd polynomial class.

Review Status

Reviewed By:
tcornwel
Date Reviewed:
1996/02/22
Test programs:
tSpecialPolynomial

Prerequisite

  • Function

Synopsis

A OddPolynomial<T> contains a set of coefficients; its fundamental operation is evaluating itself at some "x". The number of coefficients is the order of the polynomial divided by two, plus one, so is the number of available parameters.

Example

OddPolynomial<Float> pf(3); // Third order polynomial - coeffs 0 by default
pf.setCoefficient(0, 1.0);
pf[1] = 2.0; // 2x^3 + 1x^1
pf(2); // == 18

Template Type Argument Requirements (T)

  • T should have standard numerical operators. Current implementation only tested for real types (and their AutoDiffs).

Thrown Exceptions

  • Assertion in debug mode if attempt is made to address incorrect coefficients
